# PyTorch_Projects

## Pretrained model repository for Heterogeneous Architecture Models on Pytorchwith CPU/CUDA

### Dependencies 

* Pytorch 1.0
* CUDA 9.0
* Python 2.7
* Numpy 1.15
* Torchvision 0.2.1
* Matplotlib 2.2.3

**NOTE : Jetson TX2 Development kit (Nvidia Tegra X2 GPU) with Jetpack 3.3 is used for these results**
